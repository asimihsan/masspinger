masspinger by asimihsan

ICMP ping many hosts simultaneously.

Dependencies

- Boost, tested with 1.46.0 (http://www.boost.org). - ZeroMQ (http://www.zeromq.org) - log4cxx, which itself requires apr, apr-util, and xml (http://logging.apache.org/log4cxx/index.html).

Install

Windows: - Requires Microsoft Visual Studio, tested on MSVC 2010 (version 9). - Download latest version of Boost from http://www.boost.org/. Extract to e.g. c:\libs\boost\ - Follow the "Getting Started" instructions, e.g. http://www.boost.org/doc/libs/1_46_1/more/getting_started/windows.html, to build the Boost binaries. This is essentially "bootstrap" and ".\bjam" from a MSVC prompt. - The Boost binaries should now be in e.g. c:\libs\boost\stage\lib\ - - . - Ubuntu: - - sudo apt-get install liblog4cxx10 liblog4cxx10-dev, because I couldn't successfully build it from source. - g++ -I /usr/local/lib/boost_1_46_0/ -L /usr/local/lib/boost_1_46_0/stage/lib -lboost_system -lboost_program_options -Wall -ansi -pedantic ping.cpp Host.cpp Pinger.cpp -o ping (still need to set up the build system) After install run masspinger to see usage instructions. Works like e.g. "masspinger host1 host2 host3 ..."
